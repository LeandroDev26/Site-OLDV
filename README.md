# 🎻 Website Institucional: Orquestra

Este projeto é uma plataforma informativa e visualmente atrativa dedicada a uma orquestra. O objetivo principal é divulgar a história da instituição, apresentar seus membros e gerenciar a agenda de concertos, promovendo a interação com o público.

## 🚀 Funcionalidades Principais
* **Apresentação Institucional:** Seções dedicadas à missão e história da orquestra.
* **Agenda de Eventos:** Exibição de concertos e apresentações futuras.
* **Formulário de Contato:** Canal direto para comunicação com os visitantes.
* **Sistema de Matrícula:** Área restrita para novos membros (acesso via senha).

## 🛠️ Tecnologias e Estilo
* **Front-end:** HTML5, CSS3 e JavaScript.
* **Design:** Estética clássica e elegante com uso de efeito **Parallax** para uma navegação imersiva.
* **Responsividade:** Estrutura organizada para garantir leitura e navegação em diferentes dispositivos.
